涂鸦板 更新时间：2017-11-09

简介：基于html5 canvas制作，具有自由涂鸦、直线、空心矩形、空心圆、实心矩形等功能，可以对画笔自由调色。
重点：
1.需建立双层画布，第一层为绘画过程的预览层，第二层为实际画布层；
2.非全屏界面，对鼠标位置和画笔位置的获取；
3.绘画方式的判断，绝对canvas绘画函数。
