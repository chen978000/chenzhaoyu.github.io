中医科简介

中医科以弘扬中医传统文化为宗旨，以立足中医药治疗为根本，秉承大医精诚之精神，至精至微，厚德载物，悬壶济世，杏林跋涉，愿救苍生之苦，全心全意为广大患者竭诚服务。

中医科坚持发扬祖国医药遗产，博采众长，通过传承"望闻问切""辨证施治"及中药、中西医结合等中医传统诊疗方法，开展常见病、多发病及疑难杂症的诊治，形成了较强的中医特色。 主要治疗手段有：中药汤剂、药酒、药茶、针灸、推拿、理疗、中药熏洗、膏方、穴位埋线等；根据季节开展冬病夏治、冬病冬治等特色治疗。

中医科主要开展各种内科及妇科疾病的治疗，特别是心脑血管疾病、各种慢性胃炎、消化不良、肝胆疾病、久咳不愈、慢性肾病、眩晕症、盗汗失眠、高血压、高血脂及中风后遗症的调治； 情志失调（心理性）疾病、男科病、皮肤过敏疾病的中医疗法；妇科月经不调（特别是未婚女性）、带下崩漏以及一些老年性慢性疾病的饮食保健和中药调理。
