外科简介

微创手术能减少手术创伤，减轻手术给患者带来的痛苦，历来是外科医生的最高理念。 “微创”则是在当今医学发展日新月异的时代，在医学崇高理念激励下，产生的一种全新的人性化先进治疗方式。 可以这样说，与以往常规手术相比，微创外科尽可能给患者造成最小的创伤， 这种创伤包括医学结构和心理上的。其特点是经最小的小孔口，借助先进的医疗设备来达到与常规外科相同的治疗目的。 由于微创手术创伤小，患者遭受痛苦少，身体的损伤轻，康复速度快，有效减少避免了传统外科手术给患者留下的医源性创伤和合并症。

秉承“同步科技，精益求精”医疗技术创新理念的江宁新城医院，引进美国大力神微创治疗仪、电子阴道镜、C02激光机、前 列腺微波治疗仪、波姆光治疗仪、红光治疗仪等各种先进仪器设备，全力打造南京微创外科的第一品牌。 开展各项外科、妇科微创手术，治疗体表外科疾病（痣、疣、脂肪瘤、囊肿）、 各种类型痔疮、腋臭、包茎、包皮过长、乳房肿块、宫颈炎、宫颈修复、处女膜修复、阴道紧缩、子宫附件炎、子宫肌瘤等疾病。

肛肠外科

我院引进美国大力神肛肠治疗设备，能开展各种结直肠肛门手术，给痔疮患者提供了一个良好的治疗机会， 让痔疮患者不再“东奔西走”受尽病痛的折磨。该仪器在全自动电脑控制下，定位准确无论病情轻重及患病时间长短均可一次性治愈。

专业治疗：各种内痔、外痔、混合痔、肛裂、肛瘘、脱肛、肛门肿痛、便血、流脓、便秘、瘙痒、直肠炎、有异物感、下坠感等肛肠疾病。

技术优势：微创手术、不开刀、无需住院、费用低、随治随走等。

腋臭外科

江宁新城医院微创外科中心，率先采用国际领先的新一代微创大汗腺清除术，只需在腋窝开一个米粒大小的切口，将特异性大汗腺彻底清除，断绝其臭味根源，达到一次性治愈的目的。

新一代微创大汗腺清除术优势：

【优势一】米粒切口：保证腋下皮肤的完整性；

【优势二】疗效可靠：微创手术、无痛苦、随治随走、不影响工作和学习；

【优势三】专业医师：保证每例手术的质量和疗效；

【优势四】技术成熟：数千例患者成功临床经验、更彻底、更安全、不留疤痕。

术后签发疗效跟踪卡 终身疗效有保障

结石科

江宁新城医院率先引进国际最新的第Ⅲ代“可视定位爆破”高科技钬激光体外碎石设备， 是我国目前唯一“定位、聚焦”两项技术。该技术具有不开刀、不住院、无痛苦、恢复快、费用低等优点，是治疗结石病最佳选择。

诊疗范围：肾结石、输尿管结石、膀胱结石、前列腺结石等。

外伤

江宁新城医院配备有经验丰富的急诊外科医生，擅长处理常见的外伤清创缝合，骨折，石膏固定，腱鞘囊肿、体表肿物，肌腱吻合、甲沟炎、拔甲、脂肪肿块，激光除痣等。
